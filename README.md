# 🔐 Random Password Generator

I built this Random Password Generator during my **DecodeLabs Python Internship** to practice working with Python modules, functions, loops, and user input. The program creates secure passwords based on the user's choices, such as including numbers or special characters, and also gives a simple strength rating for the generated password.

## Author

**Seemab Fatima**  
Python Programming Intern – DecodeLabs

## Features

- Generate passwords of any length (minimum 6 characters)
- Choose whether to include numbers
- Choose whether to include special characters
- Make sure the password contains the selected character types
- Display a simple password strength rating
- Check for invalid input
- Generate multiple passwords without restarting the program

## Technologies Used

- Python 3
- `secrets` module
- `string` module

## ▶️ Running the Project

1. Make sure Python 3 is installed on your computer.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the program:

```bash
python password_generator.py
```

## 📖 How It Works

1. Enter the password length.
2. Choose whether to include numbers.
3. Choose whether to include special characters.
4. The program generates a random password based on your choices.
5. A simple strength rating is displayed.
6. You can generate another password or exit the program.
